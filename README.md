# Repositório das aulas de Estatística Não-Paramétrica
## Para uma amostra:
- Teste Binomial
- Teste Qui-Quadrado de Aderência
- Teste de Kolmogorov-Smirnov
- Teste de Lilliefors
- Teste de Aleatoriedade ou Iterações

## Para duas amostras:

### Relacionadas:
- McNemar
- Teste dos Sinais
- Teste de Wilcoxon
- Unilateral

### Independentes:
- Teste Qui-quadrado
- Teste exato de Fisher
- Teste U de Mann Whitney
